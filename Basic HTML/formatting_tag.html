HTML Formatting Tags

1. b => This tag is used to bold the text.

2. i => This tag is used for Italic formatting.

3. strong => This tag is used for formatting Important Text.

4. em => This tag is used for Emphasized text(to put more strees on word)

5. ins => This tag is used to define the text.

6. mark => This tag is used to Highlight the Text.

7. sub => This tag is used for subscript the text (text subscript)

8. sup => This tag is used for superscript the text (text superscript)

9. del => This tag is uesd to define Deleted or Removed text.

10. small => This tag is used for Small text formatting.
